# Governance-First Sales Prediction

End-to-end predictive analytics project applying a governance-first architecture
(Bronze, Silver, Gold layers) to forecast retail sales.

==================================================

OBJECTIVES
==================================================

- Design a scalable data pipeline
- Apply data quality and governance principles
- Build a predictive sales model
- Generate business-ready insights
- Demonstrate analytics engineering best practices

==================================================
ARCHITECTURE
==================================================

Bronze (Raw Data)
  ↓
Silver (Cleaned Data)
  ↓
Gold (Features)
  ↓
ML Model
  ↓
Dashboard

==================================================
TECH STACK
==================================================

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
- Power BI / Plotly
- GitHub

==================================================
PROJECT STRUCTURE
==================================================

data/

  └── gold/

notebooks/
src/
dashboard/

==================================================
DATASET
==================================================

This project uses the Rossmann Store Sales dataset from Kaggle.

Due to file size constraints, datasets are not stored in this repository.

Download from:
https://www.kaggle.com/c/rossmann-store-sales

==================================================
HOW TO REPRODUCE
==================================================

1. Download the dataset from Kaggle

2. Place the files in:

data/bronze/
  ├── train.csv
  └── store.csv

3. Run the notebooks in order
